# Tested on

1. MOES 1CH 10A. Built on Tuya WiFi module WA2 (WB2S) (BK7231T chip). Replacement with ESP-02S (TYWE2S). [Photo](http://git.zh.com.ru/alexey.zholtikov/zh_espnow_switch/src/branch/esp8266/hardware/MOES_1CH_10A).

```text
    Relay GPIO number               12
    Relay ON level                  HIGH
    Led pin                         4
    Led ON level                    LOW
    Internal button GPIO number     13
    Internal button trigger level   LOW
```

2. MINI 1CH 16A. Built on Tuya WiFi module WB2S (BK7231T chip). Replacement with ESP-02S (TYWE2S). [Photo](http://git.zh.com.ru/alexey.zholtikov/zh_espnow_switch/src/branch/esp8266/hardware/MINI_1CH_16A).

```text
    Relay GPIO number               13
    Relay ON level                  HIGH
    Led pin                         4
    Led ON level                    LOW
    Internal button GPIO number     3
    Internal button trigger level   LOW
    External button GPIO number     14
    External button trigger level   HIGH
```

3. LIGHT E27 SOCKET. Built on Tuya WiFi module WA2 (WB2S) (BK7231T chip). Replacement with ESP-02S (TYWE2S). [Photo](http://git.zh.com.ru/alexey.zholtikov/zh_espnow_switch/src/branch/esp8266/hardware/LIGHT_E27_SOCKET).

```text
    Relay GPIO number               12
    Relay ON level                  HIGH
    Led pin                         4
    Led ON level                    LOW
    Internal button GPIO number     13
    Internal button trigger level   LOW
```

4. TH 1CH 16A + DS18B20. Built on ITEAD WiFi module PSF-B85 (ESP8285 chip). Replacement not required. [Photo](http://git.zh.com.ru/alexey.zholtikov/zh_espnow_switch/src/branch/esp8266/hardware/TH_1CH_16A).

```text
    Relay GPIO number               12
    Relay ON level                  HIGH
    Led pin                         13
    Led ON level                    LOW
    Internal button GPIO number     0
    Internal button trigger level   LOW
    DS18B20 button GPIO number      14
```
